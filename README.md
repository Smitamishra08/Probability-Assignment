# 📊 Probability Assignment

## 📌 Overview

This repository contains solutions to various probability concepts implemented using Python.
The assignment covers both theoretical understanding and practical simulations.

---

## 🎯 Objectives

* To understand basic probability through simulations
* To apply conditional probability and Bayes’ theorem
* To work with discrete and continuous random variables
* To demonstrate the Central Limit Theorem using simulation

---

## 🛠️ Tools & Technologies Used

* Python
* NumPy
* Matplotlib
* Random module

---

## 📂 Assignment Contents

### 1️⃣ Basics of Probability

* Coin toss simulation (10,000 trials)
* Dice roll simulation (sum = 7 probability)

### 2️⃣ Probability Simulation

* Probability of getting at least one "6" in 10 rolls

### 3️⃣ Conditional Probability

* Simulation using colored balls
* Verification of conditional probability

### 4️⃣ Discrete Random Variables

* Sample generation using given probability distribution
* Mean, Variance, Standard Deviation calculation

### 5️⃣ Continuous Random Variables

* Exponential distribution simulation
* Histogram and PDF visualization

### 6️⃣ Central Limit Theorem

* Uniform distribution simulation
* Sample mean distribution visualization

---

## 📈 Key Learnings

* Experimental probability approaches theoretical probability with more trials
* Conditional probability depends on previous outcomes
* Random variables help model real-world situations
* Central Limit Theorem shows sample means follow normal distribution

1. Open the `.ipynb` file in Jupyter Notebook
2. Run all cells step by step
3. Observe outputs and visualizations
